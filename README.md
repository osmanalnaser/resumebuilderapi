# Resume Builder API

A full-stack Resume Builder backend built with Spring Boot.  
The application allows users to create, manage, and export professional resumes while providing secure authentication and payment integration for premium features.

This project is designed as a portfolio project to demonstrate modern backend development practices.

---

## Features

- JWT Authentication (Login / Register)
- Secure REST API with Spring Security
- Resume CRUD (Create, Read, Update, Delete)
- PDF Resume Export
- Razorpay Payment Integration
- Email Services with Brevo
- MongoDB Database
- Monitoring with Spring Boot Actuator
- Clean Architecture (Controller → Service → Repository)
